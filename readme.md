# Script de sauvegarde automatique

## Description
Ce script PowerShell permet d’automatiser la sauvegarde de dossiers importants.
Les données sont copiées, compressées et stockées dans une archive datée.

## Fonctionnalités
- Copie automatique de dossiers
- Compression en archive ZIP
- Sauvegarde horodatée
- Génération de logs
